# C#服务器端

## 介绍

本项目是一个基于 Socket 实现的简单服务器，可以通过 Protobuf 进行通信协议的配置，为每个客户端分发消息、处理客户端的连接和断开连接、房间管理等。

### 环境准备

- Visual Studio 2022
- .NET 9.0

### 运行方式

```
dotnet run
```

### 注意事项

- 在 Program 中修改服务器的 IP 地址（默认是 127.0.0.1）和监听端口号（默认是 1011）。
- 修改 SocketProto.proto 文件后，运行 build.bat 文件重新生成对应的 C#代码。
